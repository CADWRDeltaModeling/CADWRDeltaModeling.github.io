DSM2 version 8.2 : Exporting Channel Lengths from CSDP Network file

Created by Nicky Sandhu on Jan 27, 2021


CSDP currently has the ability to output just the channel ids and lengths.

  1. Use the Network | Export Options menu item to select only channel lengths output in station elevation format.Â 
  2. Then use Network | Save As menu to save to file which should only then have channel id and length in output.
