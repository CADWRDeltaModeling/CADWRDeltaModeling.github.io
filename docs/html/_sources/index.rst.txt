Python modules
---------------
* `vtools3 <https://cadwrdeltamodeling.github.io/vtools3/>`_: Time series functions such as godin and cosine-lanczos filters
* `schimpy <https://cadwrdeltamodeling.github.io/schimpy>`_: For working with schism input and output
* `pydsm <https://cadwrdeltamodeling.github.io/pydsm>`_: For DSM2 input/output manipulation
* `pyhecdss <https://cadwrdeltamodeling.github.io/pyhecdss/>`_: For limited access to HECDSS version 6 libraries
